# Build Guide: Add --version flag to CLI

**Issue:** JAM-256
**Purpose:** Smoke test for the `--implement-only` pipeline.

## Phase 1: Add version flag

**Objective:** Add a `--version` flag to the CLI that prints the package version.

**Changes:**
- `src/claude_engineering_agent/__main__.py` — add a `--version` argument to the
  argument parser using `parser.add_argument("--version", action="version", version="0.1.0")`.
  Place it before the mutually exclusive group so it works independently.

**Verification:**
```bash
uv run python -m claude_engineering_agent --version
# Should print: 0.1.0
```
